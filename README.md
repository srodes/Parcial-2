# Parcial 2

Resumen
- Proyecto de ejemplo para gestionar empleados con validaciones y tests JUnit 5.
- Contiene clases para modelado, gestión, catálogo de cargos y constantes de mensajes, además de pruebas unitarias y un documento de testing.

Autor y versión
- Autor: Sergio Rodes
- Versión: 1.0

Estructura de ficheros (resumen)
- Empleado.java — modelo del empleado (nombre, cargo, salario).
- Empleados.java — gestor de la colección de empleados (alta con validación, aumentarSalario, mostrarListado).
- Cargos.java — lista y validación de cargos profesionales.
- Mensajes.java — constantes de textos y números mágicos (prompts y convenio salarial).
- SistemaGestionEmpleados.java — aplicación principal (método main).
- TESTING.md — análisis de caja negra.
